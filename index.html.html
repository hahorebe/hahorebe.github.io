Featured Projects

★ Featured Project

01 — IoT + Hardware

AquaFlow Smart

Automatic water flow control system for Filipino households. Uses float sensors, relay modules, and solenoid valves to automatically stop water supply when the tank is full and resume when it drops — no internet or app required. Built for Malaybalay City communities.

ESP32 IoT Sensors Electronics ₱1,200–₱2,500/unit

02 — IoT + Web System

Smart Attendance System

Students tap their RFID card on an ESP32 + RC522 reader and their attendance is automatically marked and instantly updated on the teacher's web dashboard. Eliminates manual attendance taking entirely.

ESP32 RC522 RFID PHP XAMPP Bootstrap CSS

03 — Web Platform

Rice Connect

A web platform connecting rice farmers and buyers directly — cutting out the middleman so Filipino farmers earn more. Buyers can post offers, sellers view prices and contact buyers, and admins manage the platform efficiently.

PHP XAMPP Bootstrap CSS MySQL
